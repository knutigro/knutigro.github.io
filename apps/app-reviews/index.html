Welcome to The project page for App Reviews.

App Reviews make it super simple for developers and other interested to keep track of new user reviews to iPhone apps. App Reviews run in your Mac-OSX statusbar and notifies you when there are new reviews.

If you like App Reviews and use it, please make a Donation for the hours I have spent getting this app up and running. I am also happy to receive feature requests, issues, or pull requests on the Open Source GitHub pages.
